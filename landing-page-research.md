# AI Image Tool Landing Page Research

**Research Date:** 2026-02-06
**Purpose:** Inform VixPic landing page positioning

---

## Executive Summary

After analyzing TypingMind, Krea.ai, Playground.com, CodeGPT, and other BYOK tools, the key insights are:

1. **LTD (Lifetime Deal) works** - TypingMind's success (20,641+ customers at $39-$99) proves the model
2. **BYOK is a differentiator** - Position as "Full Cost Control" + "No Rate Limits" + "Complete Data Ownership"
3. **Free tier is essential** - All tools offer generous free usage to hook users
4. **Social proof is crucial** - User counts, review badges, testimonials with photos
