### 3. Horizontal SaaS Massive Headwinds (2026 Prediction)
> "Bootstrapped horizontal SaaS will experience massive headwinds in 2026"

**Why:**
- Venture money flooding big horizontal markets
- AI-first companies creating noise
- Competition in content/SEO making marketing harder
- Plateaus at $500K-$2M ARR, hard to break through

**Exceptions:**
- Lifestyle projects on single traffic source (SEO) can reach $20-40K/mo
- Vertical or orthogonal positioning avoids the worst headwinds

**VixPic Implication:** BYOK positioning is **orthogonal** - not competing directly with Midjourney/DALL-E but serving different buyer (API key owners). This is a strength.

## 17:00 UTC - Build Session: Image Generation UI

### What Was Built

**Core App Structure:**
- `/generate` page - Full image generation interface
- `/settings` page - API key management
- TypeScript types for providers, models, generation params
- Local storage hooks for API keys, history, preferences

**Generate Page Features:**
- Provider/model selection with 4 providers (Replicate, FAL, OpenAI, Together)
- 8+ models with real cost estimates (FLUX Schnell ~$0.003, DALL-E 3 ~$0.04)
- Aspect ratio controls (1:1, 16:9, 9:16, 4:3, 3:4)
- Advanced settings: steps, guidance scale, negative prompt
- Generation history display with thumbnail grid
- Cost estimate before generation

**Settings Page Features:**
- Secure API key management (local storage only)
- Show/hide key toggle
- Provider documentation links
- Usage statistics (images generated, total cost, savings)
- Clear history option

**UI Components Added:**
- input, textarea, select, tabs, dialog, label, slider, switch, badge

### Architecture Decisions

1. **100% Client-Side:** Everything runs in browser, no backend needed for MVP
2. **localStorage:** API keys never leave device - security by design
3. **Provider Abstraction:** Easy to add new providers via types.ts
4. **Cost Tracking:** Built-in from day one for value proposition

### What's Next

2. **Edge Proxy:** Vercel Edge Functions for CORS handling
3. **Image Download:** Proper save functionality
4. **Error Handling:** Rate limits, invalid keys, generation failures
